What are <meta> elements in html? Where are they used? Provide two examples.

A meta element is data that describes the documents author, keyword or description. It is used within the html document and will not be shown on the actual page. Two examples are meta name = description and meta charset= "UTF-8.

Briefly describe the correct usage of the following HTML5 semantic elements: <header>, <article>, <section>, <footer>. Provide an example of each.

  1. Header is used as a header for the document. Example: Reasons why HTML is awesome.
  2. Article is for independent text. Example: Apples are the best fruit.
  3. Section is used for sections of the document such as chapters and footers. Example: <section> <h1>Cookies</h1> </section>
  4. Footer is used for the footer for the document. Example: ©Allen Clark Nelson Jr.
